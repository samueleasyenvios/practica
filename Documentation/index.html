Introduction

Minton React is a React with Typescript + Redux based admin template created using Create React App, Redux & Bootstrap v5.1.3. Minton React is crafted by following the best industry standards in order to provide speed and scalability. The clean code structure allows you to build any React-based web application without any hustle. The theme includes beautiful dashboards, many ready-to-use hand-crafted components, widgets, pages, etc. The components can be used very easily on any page. It is fully responsive and customizable.

Additional features :

  • Lazy Loading(React.lazy)
  • Localization(i18n)
  • Custom Hooks

This documentation guide is for all users who can even be beginners or experienced react developers.
